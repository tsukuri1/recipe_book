Основные экраны: авторизация, главная, каталог, карточка рецепта, добавление рецепта, список покупок, профиль, настройки, админ-панель.

## Описание экранов

### Авторизация

Экран содержит вход и регистрацию. Для демонстрации предусмотрены тестовые аккаунты пользователя и администратора. После входа приложение получает профиль и роль.

### Каталог

Каталог показывает карточки рецептов, поиск и категории. Карточка должна быть достаточно крупной, чтобы пользователь мог быстро оценить блюдо по фото и названию.

### Карточка рецепта

Детальный экран содержит изображение, описание, время приготовления, порции, сложность, КБЖУ, ингредиенты и шаги. Отсюда пользователь добавляет ингредиенты в покупки.

### Добавление рецепта

Форма собирает структурированные данные рецепта. Ингредиенты вводятся с количеством и единицей измерения, чтобы список покупок был полезным.

### Админ-панель

Администратор видит статистику, жалобы и рецепты на проверке. Это отдельный сценарий, доступный только роли `ADMIN`.

## Проверка UX

Для проверки UI нужно пройти основные сценарии: вход, поиск рецепта, открытие карточки, добавление в покупки, редактирование профиля, сохранение настроек и модерация рецепта.
