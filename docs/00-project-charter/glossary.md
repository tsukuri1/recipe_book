# Глоссарий

| Термин | Описание |
|---|---|
| Рецепт | Карточка блюда с фото, ингредиентами, шагами и КБЖУ |
| КБЖУ | Калории, белки, жиры и углеводы |
| Список покупок | Перечень ингредиентов, необходимых для приготовления |
| Администратор | Пользователь, который модерирует рецепты и жалобы |
| PCMEF | Архитектурное разделение Presentation, Control, Mediator, Entity, Foundation |

## Дополнительные термины

| Термин | Описание |

## Роль глоссария

Глоссарий нужен для единообразия терминов в коде, документации и пояснительной записке. Например, термин «рецепт» в проекте означает не просто текстовое описание блюда, а карточку с ингредиентами, шагами, изображением, категорией, сложностью и КБЖУ. Это помогает связывать требования, диаграммы и реализацию без разночтений.
