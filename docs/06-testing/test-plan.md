| Android build | `./gradlew :app:assembleDebug` | BUILD SUCCESSFUL |

## Ручные сценарии Android

### Сценарий 1. Вход пользователя

1. Открыть приложение.
2. Ввести `user@ladushki.app` и пароль `1234`.
3. Нажать кнопку входа.
4. Проверить, что открылась главная и отображается приветствие.

### Сценарий 2. Создание рецепта

1. Перейти на экран добавления.
2. Заполнить название, категорию, время, порции, ингредиенты и КБЖУ.
3. Выбрать фотографию.
4. Сохранить рецепт.
5. Проверить сообщение о том, что рецепт отправлен на проверку.

### Сценарий 3. Список покупок

1. Открыть карточку рецепта.
2. Добавить ингредиенты в покупки.
3. Перейти в список покупок.
4. Отметить часть продуктов купленными.
5. Очистить купленные позиции.

### Сценарий 4. Админ-панель

1. Войти под `admin@ladushki.app`.
2. Открыть профиль и перейти в админ-панель.
3. Проверить статистику, жалобы и рецепты на проверке.
4. Одобрить или отклонить рецепт.

## Критерии завершения тестирования

Тестирование считается достаточным для курсового проекта, если backend-тесты проходят, покрытие выше 40%, а ручные сценарии подтверждают работу основных экранов мобильного клиента.
