# Оценка трудозатрат

| Показатель | Значение |
|---|---|
| Тип проекта | Учебное мобильное приложение |
| Размер | Малый |
| Основные трудозатраты | UI, состояние, документация, API |
| Оценка | 2-4 недели учебной работы |

## Обоснование оценки

Проект относится к учебным мобильным приложениям малого размера, но включает несколько технологических областей: Android UI, backend, база данных, безопасность, REST API, тесты и документация. Поэтому трудозатраты выше, чем у простого одностраничного приложения.

## Распределение трудозатрат

| Область | Доля | Комментарий |
|---|---:|---|
| Анализ и требования | 15% | сценарии, use case, глоссарий |
| Архитектура и БД | 20% | PCMEF, ER, API, ADR |
| Android-клиент | 30% | экраны, состояние, навигация, Room |
| Backend | 20% | контроллеры, сервисы, JWT, JPA |
| Тесты и документация | 15% | JUnit, JaCoCo, README, отчёт |

## Вывод

Оценка 120 часов является реалистичной для курсового проекта мобильной траектории, если часть работы выполняется параллельно: UI, backend и документация постепенно уточняют друг друга.
